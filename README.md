🕌 Prayer Times App

A cross-platform mobile and web utility application built with modern web technologies. It provides accurate daily prayer times, media playback for various Ezan recordings, and digital Islamic resources.

✨ Features

🕒 Accurate Prayer Times: City-specific and daily prayer times.

🔊 Audio Assets: Built-in audio playback including Ayasofya and Medine Ezan.

🕋 Islamic Resources: Includes Esma-ül Hüsna, holidays, and daily religious tools.

📱 Cross-Platform: Developed using Angular and Ionic with Capacitor support for Android.

🛠️ Tech Stack

⚡ Framework: Angular

🎨 Mobile & UI: Ionic Framework

🤖 Native Integration: Capacitor (Android)

🚀 Installation & Setup

Clone the repository:

git clone https://github.com/emin129/prayer-times-app.git

Navigate to the project folder:

cd prayer-times-app

Install dependencies:

npm install

Run the application locally:

ionic serve

📱 Build for Android

To run and compile the application on an Android device:

Add the Android platform:

npx cap add android

Sync the assets and code:

npx cap sync

Open Android Studio:

npx cap open android
